Tools and Architecture Set Up
In this activity, the project was set up by preparing the tools and system structure. The  was used to open the project and create a new folder for all files. This folder serves as the main directory of the system.
A local Git repository was also created to track changes in the project. This helps in saving different versions and managing updates easily.
A README file was made to explain the purpose of the project. It also describes the folder structure and how the system is organized.
The folder structure is divided into parts like components, pages, services, and database. This makes the project clean and easy to understand.
The system uses a simple layered structure. The presentation layer is for the user interface, the logic layer handles the process, and the data layer manages the database. This helps make the system organized and easier to maintain.
